## ğŸ†˜ Common Issues

### "Access Denied" when logging in as admin
**Fix**: Your account is not set up as admin. Contact IT/Developer to add your user ID to the admin list.

### "No pending registrations"
**Meaning**: All users have been approved or rejected. New sign-ups will appear here.

### "No questionnaire responses"
**Meaning**: No users have completed assessments yet, or data is still loading.

### Can't see any users in LSN management
**Meaning**: No users have been approved yet. Approve users first in "Validate Registrations".

### App crashes or freezes
**Fix**: 
1. Force close the app
2. Clear app cache (Settings > Apps > MindCare > Clear Cache)
3. Restart app
4. If persistent, contact developer
